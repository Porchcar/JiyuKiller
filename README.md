## 简介
JiyuKiller 可帮助你脱离极域课堂管理软件的控制，恢复系统正常使用权限。

## 环境要求
- 系统：Windows 7 及以上
- Python 版本：3.9 及以上

## 快速开始
### 1. 从 Releases 下载（推荐）
[前往 Releases 下载最新版](https://github.com/Porchcar/JiyuKiller/releases)

### 2. 克隆仓库运行
```bash
git clone https://github.com/Porchcar/JiyuKiller.git
cd JiyuKiller
python JiyuKiller.py
